# 장바구니 기반 추천 시스템 프로젝트

## 📌 프로젝트 개요

고객의 장바구니 데이터를 활용하여 **함께 구매될 가능성이 높은 상품을 추천**하고, **번들 제안**을 통해 객단가(AOV)를 높이는 추천 시스템 개발 프로젝트입니다.

- 장바구니 데이터 기반 상품 추천 규칙 도출

- 데이터 기반 의사결정을 위한 인터랙티브 대시보드 구축
- 실제 비즈니스에 적용 가능한 추천 시스템 구현
